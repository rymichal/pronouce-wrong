
Creates a video from a string argument. 
- Displays the string text in a video.
- Fetches and adds an azure text-to-speech voice to the video. 


Setup: 
add the following to an .env
  AZURE_SPEECH_KEY
  AZURE_SPEECH_REGION

Example Usage: 
dotnet run 'The German Baking Book'
